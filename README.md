# Royal Run 🎮

Royal Run is a **3D endless runner game** developed in **Unity using C#**, where the player continuously runs through an endless environment, avoids obstacles, collects coins, and aims for the highest possible score.

## 📌 Features
- **Procedural Level Generation** using reusable level chunks
- **Smooth Player Movement** with input-based controls
- **Lane Switching System** for left/right movement
- **Obstacle Spawning** and automatic destruction for optimization
- **Coin Collection System**
- **Real-time Score Tracking**
- **Physics-based Collision Handling**
- **Fence & Trigger Animations**
- **Game Over System**
- **Visual Post Processing Effects**
- **Prefab Variants & Asset Management**
- **Efficient Chunk Handling using C# Lists**

## 🛠️ Technologies Used
- **Unity Engine**
- **C#**
- **Unity Physics**
- **Prefab System**
- **Post Processing**
- **Input Actions**

## 🎯 Gameplay
The player runs automatically through an endless generated path.  
The objective is to:
- Avoid obstacles
- Collect coins
- Score as high as possible
- Survive for as long as possible

As the game progresses, the challenge increases, testing the player’s reflexes and timing.
